# OpenCosmos

A creative platform built on a simple recognition: we are not separate from the universe we inhabit.

Our purpose: to reduce suffering, nourish flourishing, and enable acts of wisdom.

---

The working code currently lives at:

- **[opencosmos](https://github.com/shalomormsby/opencosmos)** — Product applications, knowledge corpus, and Cosmo AI
- **[opencosmos-ui](https://github.com/shalomormsby/opencosmos-ui)** — Design system (`@opencosmos/ui`)

These repositories will migrate to this organization as the platform matures.
